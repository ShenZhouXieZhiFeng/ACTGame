Process to start the game with source code in Unity 5:
1. Open the project with Unity 5.
2. Run the items below "Custom Editor" one by one to pack the configuration files and resources.
3. Run the "hfs.exe" in "Tool" directory.
4. Set up the port num as the file "GlobalDef.cs" s_FileServerPort write.
5. Drag the sub-folder of "ExportedAssets", file "crossdomain.xml", and file "ResVer" to HFS.
6. Run the game in Unity 5, than you can enjoy it.
